# Preferências gerais

- Textos exibidos para usuários finais devem ser escritos em português do Brasil, com acentuação correta.

# Validação visual

- Para mudanças estritamente visuais ou de experiência de interface, deixe a validação final para o usuário. Não é necessário iniciar dev server, abrir navegador, tirar screenshots ou rodar comandos apenas para conferir aparência, salvo quando o usuário pedir explicitamente ou quando a mudança também envolver risco funcional, build, roteamento, acessibilidade crítica ou comportamento interativo que precise ser verificado pelo agente.
